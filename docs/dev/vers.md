# 版本号规范
`a.b.c(d)-e`</br>
- `a`：主版本号，大更新后修改</br>
- `b`：子版本号，新增/兼容功能后修改</br>
- `c`：修正版本号，修复bug后修改</br>
- `d`：编译版本号，提交次数修改，默认为0000，[更多格式](dev/vers.md?id=编译版本号更多格式)</br>
- `e`：版本分类，详情请看[版本分类](dev/vers.md?id=版本分类)</br>

## 版本分类

### 正式版
release：正式上线版本，稳定版

### 开发版
beta：开发使用版本，dev开发频道最先更新

## 编译版本号更多格式
`(abcd)`</br>
- `a`： 提交为当月</br>
- `b`： 提交次数百位数</br>
- `c`： 提交次数十位数</br>
- `d`： 提交次数个位数</br>

### 备注
十月：Ⅹ
十一月：Ⅺ
十二月：Ⅻ

### 示例
`(X007)`
